4.  Click **Add Integration** and search for **Unii Alarm**.
5.  Enter your Alarm IP, Port (6502), and Shared Key.

## Features
*   **Arm Away**: Arms Section 1.
*   **Disarm**: Disarms Section 1.
*   **Status**: Polls the alarm status every 30 seconds.
